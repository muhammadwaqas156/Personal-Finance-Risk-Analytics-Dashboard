# 📊 Personal Finance & Risk Analytics Dashboard

## 📌 Project Overview

This project showcases an end-to-end **Personal Finance & Risk Analytics Dashboard** built using Power BI. It analyzes user-level financial data to uncover insights into **income, spending behavior, savings performance, financial risk, and behavioral patterns**.

The dashboard simulates a real-world analytics solution used in **banking, fintech, and financial advisory domains**, helping stakeholders make informed decisions based on data-driven insights.

---

## 🎯 Objectives

* Evaluate overall financial health using income and expense analysis
* Identify key spending patterns and high-cost categories
* Measure savings performance and goal achievement
* Assess financial risk using credit and debt indicators
* Analyze behavioral patterns affecting financial decisions

---

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard development and visualization
* **DAX (Data Analysis Expressions)** – Calculated measures and KPIs
* **Power Query** – Data cleaning and transformation
* **Data Modeling** – Relationship building and calendar table

---

## 📂 Dataset Description

The dataset contains structured financial records with the following attributes:

* Monthly income and expenses
* Savings, savings rate, and financial goals
* Credit score and debt-to-income ratio
* Loan payments and investment amounts
* Fraud indicators
* Spending categories (e.g., Groceries, Utilities, Healthcare)
* Financial stress levels and behavioral metrics
* Transaction counts and subscription services

## 📈 Key Metrics (DAX Measures)

Key calculated measures used in this project:

* **Net Income** = Income – Expenses
* **Savings Rate %** = Total Savings / Total Income
* **Savings Efficiency %** = Total Savings / Net Income
* **Fraud Cases** = Sum of fraud flags
* **Fraud Rate %** = Fraud Cases / Total Records
* **Expense Ratio** = Total Expenses / Total Income
* **Savings Gap** = Net Income – Actual Savings

These metrics provide a deeper understanding of both **financial performance and user behavior**.

---

## 🔍 Key Insights

* Users with higher **debt-to-income ratios** tend to experience higher financial stress
* A gap exists between **net income and actual savings**, indicating inefficient saving behavior
* **Discretionary spending** contributes significantly to total expenses
* Salaried individuals show more **consistent savings patterns** compared to freelancers
* Fraud cases are relatively low but concentrated in specific categories and user segments
